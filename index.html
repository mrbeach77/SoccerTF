SoccerTF

This project is a full Next.js application. Opening this file directly will only show a static preview, so below is an interactive demo and the exact steps needed to run the complete web app with maps, trainer profiles, and dashboards.

Run the full app locally

Install dependencies and start the dev server to access all features.

npm install
npm run dev
Open http://localhost:3000

What you get

Map + list search, trainer profiles, privacy-first contact links, and admin scaffolding are all enabled when the app is running.

Need help?

See README.md for environment setup, Supabase configuration, and optional seed data to unlock full functionality.
